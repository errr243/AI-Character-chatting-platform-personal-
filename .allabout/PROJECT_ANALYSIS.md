# AI 캐릭터 채팅 플랫폼 분석 보고서

## 1. 프로젝트 개요

이 프로젝트는 Next.js와 TypeScript로 구축된 **Gemini 기반 AI 캐릭터 채팅 플랫폼**입니다. 사용자는 이 플랫폼을 통해 AI 캐릭터와 대화할 수 있으며, 점진적 웹 앱(PWA)으로 설계되어 모바일 환경에서도 앱처럼 사용할 수 있습니다.

주목할 만한 특징은 **사용자 데이터가 서버에 저장되지 않는다**는 점입니다. 채팅 기록, 캐릭터 설정 등 모든 주요 사용자 콘텐츠는 서버 측 데이터베이스가 아닌, 사용자의 브라우저 저장소(`localStorage`)에 직접 보관됩니다.

## 2. 아키텍처

현대적인 Jamstack 아키텍처를 따르며, Next.js 프론트엔드와 BFF(Backend-for-Frontend) API 패턴을 사용합니다.

- **프론트엔드**: Next.js App Router 기반으로 구현되었으며, 사용자 인터페이스와 클라이언트 측 로직을 담당합니다.
- **백엔드 API (`app/api`)**: 사용자 데이터 저장을 제외한 서버 환경이 필요한 작업들을 처리합니다.
    - **AI 채팅**: Google Gemini API 요청을 중계하여 API 키를 안전하게 보호합니다.
    - **인증**: NextAuth.js를 통해 사용자 인증(로그인)을 관리합니다.
    - **결제**: Stripe 구독 모델을 통합하고, 웹훅(Webhooks)을 통해 접근 권한을 관리합니다.
- **데이터 저장소**: 핵심 사용자 데이터(채팅 기록, 캐릭터, 설정)는 모두 브라우저의 `localStorage`에 저장됩니다. 이는 서버리스(Serverless) 데이터 관리 방식이며, 애플리케이션을 가볍고 빠르게 만들어줍니다.

## 3. 핵심 기능

- **AI 채팅 (`/chat`)**: 사용자가 AI 캐릭터와 실시간으로 대화하는 핵심 기능입니다.
- **캐릭터 관리 (`/characters`)**: 사용자가 자신만의 AI 캐릭터를 생성하고, 성격 및 기타 특성을 정의할 수 있습니다.
- **채팅 기록 (`/history`)**: 과거 대화 내용을 조회하고 다시 볼 수 있습니다.
- **AI 유틸리티**:
    - **텍스트 요약 (`/api/summarize`)**: 긴 텍스트를 요약하는 기능입니다.
    - **번역 (`/translate`)**: 텍스트를 다른 언어로 번역하는 기능입니다.
- **수익 모델**: Stripe를 이용한 구독 기반의 유료화 모델이 포함되어 있습니다.
- **사용자 인증**: NextAuth.js를 통해 사용자가 안전하게 로그인하고 자신의 계정을 관리할 수 있습니다.
